Задача №4

  • Створіть функцію, яка повертає результат ділення числа a на число b з додаванням рядка "Результат ділення: "
  • Викличте функцію передаючі різні значення, у тому числі не передаючи зовсім. Функція не має повертати NaN, Infinite або помилку
Відповідь:
function divide(a, b) {
// Перевірка на передані значення і ділення на нуль
if (typeof a !== 'number', typeof b !== 'number', isNaN(a), isNaN(b), b === 0) {
return "Результат ділення: некоректні дані";
}

// Виконання ділення
let result = a / b;

// Повернення рядка з результатом
return "Результат ділення: " + result;

// Приклади виклику вункції з різними значеннями
console.log(divide(10, 2)); - Результат ділення: 5
console.log(divide(10, 0)); - Результат ділення: некоректні дані
console.log(divide(10)); - Результат ділення: NaN
console.log(divide()); - Результат ділення: NaN
console.log(divide(15, 3)); - Результат ділення: 5
console.log(divide("a", 3)); - Результат ділення: NaN
